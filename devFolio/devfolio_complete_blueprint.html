Subdomain routing trick
How username.devfolio.tech works
In Express, read req.hostname → extract username before the dot → query MongoDB for that user → render their portfolio. Set wildcard DNS *.devfolio.tech pointing to your server. This is the trickiest part — do it in week 3.
